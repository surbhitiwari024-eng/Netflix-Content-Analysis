# Netflix-Content-Analysis
Netflix content analysis using data analytics
Netflix Content Analysis Using Data Analytics
________________________________________
 Project Overview
This project analyzes Netflix’s Movies and TV Shows dataset to understand content distribution, growth trends, genre popularity, audience targeting, and geographical focus. The objective is to derive meaningful business insights using data analytics techniques and present them in a clear and structured manner.
________________________________________
 Objectives
•	Analyze the distribution of Movies and TV Shows on Netflix
•	Study content growth trends over the years
•	Identify popular genres and content ratings
•	Understand country-wise content production
•	Derive strategic insights from data
________________________________________
Dataset
•	Source: Netflix Movies and TV Shows Dataset (Kaggle)
•	Description:
The dataset contains information such as title, type, country, release year, date added, rating, duration, genre, and description of Netflix content.
________________________________________
Tools & Technologies Used
•	Python – Data cleaning and preprocessing
•	SQL – Data analysis and querying
•	Power BI (Conceptual Design) – Dashboard planning and visualization logic
•	GitHub – Project repository and version control
________________________________________
Key Insights
•	Movies dominate Netflix’s content library, while TV shows play a key role in audience retention
•	Content addition increased significantly after 2016, indicating aggressive expansion
•	The United States and India are major contributors to Netflix’s content catalog
•	Drama and Comedy are the most popular genres
•	Netflix primarily targets mature audiences through TV-MA rated content
________________________________________
Conclusion
This project demonstrates how data analytics can be applied to real-world business scenarios. By combining Python, SQL, and visualization concepts, the analysis provides valuable insights into Netflix’s content strategy and audience focus.
